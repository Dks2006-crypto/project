    GitHub - это

    это облачная платформа для хостинга IT-проектов и совместной разработки, под капотом которой находится популярная система контроля версий Git, а также полноценная социальная сеть для разработчиков.

    GitBush - это

    это интерфейс командной строки (CLI), который объединяет функциональность Git и Bash, популярной оболочки Unix.Он предлагает удобный способ взаимодействия с репозиториями Git, выполнения задач контроля версий и навигации по файловой системе.
